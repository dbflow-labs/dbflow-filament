# DBFlow Filament 1.0.0

First stable release of the standard Filament integration for DBFlow Core.

## Highlights

- Requires `dbflowlabs/core:^1.0`
- No breaking API changes from `1.0.0-rc.2`
- My Workflow Tasks, instances, definition editor, and timeout fields aligned with Core `1.0.0`

## Upgrade

```bash
composer require dbflowlabs/filament:^1.0
```

Pair with `dbflowlabs/core:^1.0`. See Core [UPGRADE-1.0.md](https://github.com/dbflow-labs/dbflow-core/blob/1.0.0/UPGRADE-1.0.md).
